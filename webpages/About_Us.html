About This Tool

Backend data processing

How did we handle missing data points?
By using a package within scikit-learn's regression function set, we used a polynomial regression to predict and replace the mising data points from a number of different installations across the state.

Interacting with PVWatts

We used the PVWatts API for a lot of our user-facing coverage. For our calls to PVWatts we used optimal panel installation and orientation conditions to provide a reference point for what a "perfect" cell would achieve.
